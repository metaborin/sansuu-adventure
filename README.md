# さんすうアドベンチャー 1ねんせい 🎒

小学1年生が、算数を **順を追って楽しく** 学べるブラウザゲームです。
1回あたり 3〜5分 で遊べるミニゲームを、ステージ1〜14として用意しています。

- 文字は少なめ・ひらがな中心
- ボタンは大きく丸く、間違えても否定的な言葉は出さない
- 進捗は自動で保存（`localStorage`）
- スマホ / タブレット / PC 対応のレスポンシブ
- 外部APIなし。ローカルだけで完結

React + TypeScript + Vite で作られています。

### ✨ 学びやすさの くふう

- **段階ヒント**：まちがえるたびに、ヒントがだんだん くわしくなります（答えはすぐに出しません）。
- **つまり防止のみちびき**：2回まちがえると、正解のボタンがやさしく光って助けます（採点は「1回めの正解」だけを数えるので、光ってもスコアには影響しません）。
- **難易度の自動調整**：直近の成績で、そのステージの出題レベル（1〜3）が自動で上下します。全問正解でレベルアップ、2問以下でレベルダウン。ステージ5「かずの ならび」は 1〜20 → 1〜50 → 1〜100 のように、少しずつ難しくなります。
- **たのしい演出**：問題ごとにイラストが ぽんっと登場、クリアでスターや紙吹雪。
- **音（BGM・効果音・読み上げ）**：やさしいBGM、正解・不正解・クリアの効果音、そして問題文の読み上げ（まだ字が読めない子むけ）。
  - BGMと効果音は音声ファイルを使わず **Web Audio API** でその場で合成、読み上げはブラウザ内蔵の **Web Speech API** を使うので、**外部通信ゼロ・オフラインでも動作**します（アセット追加なし）。
  - **BGM・効果音**：はじめから**オン**。画面右上（ホーム）や問題画面の 🔊 ボタン、ホーム下の「おうちの人・せんせいむけ」から**オン/オフ**でき、設定は保存されます（教室で無音にしたいとき用）。
  - **読み上げ**：はじめは**オフ**。必要なときに、問題文の横の 🔊 ボタンで**その都度 読み上げ**できます。保護者向けに「じどう よみあげ」をオンにすると、問題が出るたびに自動で読み上げます。
  - ※読み上げは端末に日本語音声が入っている場合に有効です（Windows/Mac/iOS/Android は標準で入っていることが多いです）。

Node.js（18以上を推奨。開発は Node 24 で確認）が入っている状態で、
このフォルダで以下を実行してください。

## 🎮 あそびかた

1. ホーム画面で遊びたいステージのカードを押す（最初はステージ1だけ解放）。
2. 1ステージ＝**5問**。えらんで答える。
   - 正解 → 「よくできたね！」＋「つぎへ」
   - まちがい → **答えは出さず**、やさしいヒントを表示（何回でもやり直せる）
3. **4問以上正解でクリア** → 次のステージが解放されます。
4. クリア画面でスター（最大3つ）を獲得。

### せんせい・ほごしゃ向け

- ホーム下部の **「🔓 すべて かいほう」** … 全ステージをすぐ解放
- **「🔄 しんちょくを リセット」** … 進捗を初期化

## ➕ あたらしい問題・ステージの ふやしかた

問題を追加しやすい構造にしています。

- **同じステージに問題バリエーションを足す**
  `src/questions/stageXX.ts` の生成関数に分岐を追加するだけ。
- **新しいステージを足す**
  1. `src/data/stages.ts` の配列に1件追加
  2. `src/questions/stageXX.ts` を作って生成関数を書く
  3. `src/questions/index.ts` の `GENERATORS` に1行登録
- **新しい見た目（イラスト）を足す**
  `src/types.ts` の `Visual` に種類を1つ追加し、`src/components/Visual.tsx` に描画を追加。

## 🔊 音声について

音声は [src/utils/audio.ts](src/utils/audio.ts) に **1モジュールで集約**しています。

- BGM・効果音は Web Audio API（オシレーター）で合成、読み上げは Web Speech API。どちらも外部ファイル・外部通信なし。
- 設定は2種類（それぞれ localStorage に保存）：
  - `sfx`（BGM＋効果音、デフォルト**オン**）… `setSfxEnabled()` / `isSfxEnabled()`
  - `speech`（自動読み上げ、デフォルト**オフ**）… `setSpeechEnabled()` / `isSpeechEnabled()`
- 呼び出し側は `playTap()` `playCorrect()` `playWrong()` `playClear()` `startBgm()` `speak(text)`（手動・常に鳴る）/ `speakAuto(text)`（設定オン時のみ）を呼ぶだけ。
- BGMはブラウザの自動再生制限に合わせ、**最初のタップ（ユーザー操作）で開始**します（`App.tsx` の `pointerdown` リスナー）。
- 将来 **本物の効果音ファイルや別の音声エンジンに差し替えたい**ときは、`audio.ts` の各関数の中身を変えるだけでOK（呼び出し側は変更不要）。

## 💾 データ（進捗）について

`localStorage` のキー `sansuu-adventure-1nensei:progress` に、
`Progress` 型（`src/types.ts`）で保存しています。
データ形式を変更したら `src/utils/storage.ts` の `PROGRESS_VERSION` を上げると、
古いデータを安全にリセットします。
