# さんすうアドベンチャー 1ねんせい 🎒

小学1年生が、算数を **順を追って楽しく** 学べるブラウザゲームです。
1回あたり 3〜5分 で遊べるミニゲームを、ステージ1〜14として用意しています。

- 文字は少なめ・ひらがな中心
- ボタンは大きく丸く、間違えても否定的な言葉は出さない
- 進捗は自動で保存（`localStorage`）
- スマホ / タブレット / PC 対応のレスポンシブ
- 外部APIなし。ローカルだけで完結

React + TypeScript + Vite で作られています。

---

## 🚀 じっこう方法（実行方法）

Node.js（18以上を推奨。開発は Node 24 で確認）が入っている状態で、
このフォルダで以下を実行してください。

```bash
npm install
npm run dev
```

表示された `http://localhost:5173/` をブラウザで開くと遊べます。

### そのほかのコマンド

```bash
npm run build     # 本番用にビルド（型チェックも実行）
npm run preview   # ビルド結果をローカルで確認
```

## 🎮 あそびかた

1. ホーム画面で遊びたいステージのカードを押す（最初はステージ1だけ解放）。
2. 1ステージ＝**5問**。えらんで答える。
   - 正解 → 「よくできたね！」＋「つぎへ」
   - まちがい → **答えは出さず**、やさしいヒントを表示（何回でもやり直せる）
3. **4問以上正解でクリア** → 次のステージが解放されます。
4. クリア画面でスター（最大3つ）を獲得。

### せんせい・ほごしゃ向け

- ホーム下部の **「🔓 すべて かいほう」** … 全ステージをすぐ解放
- **「🔄 しんちょくを リセット」** … 進捗を初期化

## ➕ あたらしい問題・ステージの ふやしかた

問題を追加しやすい構造にしています。

- **同じステージに問題バリエーションを足す**
  `src/questions/stageXX.ts` の生成関数に分岐を追加するだけ。
- **新しいステージを足す**
  1. `src/data/stages.ts` の配列に1件追加
  2. `src/questions/stageXX.ts` を作って生成関数を書く
  3. `src/questions/index.ts` の `GENERATORS` に1行登録
- **新しい見た目（イラスト）を足す**
  `src/types.ts` の `Visual` に種類を1つ追加し、`src/components/Visual.tsx` に描画を追加。

## 💾 データ（進捗）について

`localStorage` のキー `sansuu-adventure-1nensei:progress` に、
`Progress` 型（`src/types.ts`）で保存しています。
データ形式を変更したら `src/utils/storage.ts` の `PROGRESS_VERSION` を上げると、
古いデータを安全にリセットします。
